## install nvm

```
curl -o- 
https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash

nvm install v12.16.1

npx create-react-app nexmo_client_sdk_react_tutorial

cd nexmo_client_sdk_react_tutorial

echo "v12.16.1" > .nvmrc
```
